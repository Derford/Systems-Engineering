# Стейкхолдеры

| Стейкхолдер | Цель | Ожидания |
|---|---|---|
| Студент | найти стажировку или работу | релевантные рекомендации, простой отклик, понятные статусы |
| Работодатель | найти кандидатов | качественные отклики, фильтрация, удобное назначение интервью |
| Карьерный центр | управлять трудоустройством | отчёты, контроль качества вакансий, аналитика |
| Модератор | поддерживать качество площадки | очередь проверки, инструменты блокировки, история действий |
| Университет | повысить показатели трудоустройства | прозрачная статистика, интеграция с ЛК студента |
| Администратор | поддерживать систему | мониторинг, роли доступа, аудит |

## Конфликты интересов

- Работодателю хочется быстро публиковать вакансии, но карьерному центру нужна модерация.
- Студент хочет получать больше рекомендаций, но система должна сохранять приватность данных.
- Университету нужна аналитика, но отчёты должны быть обезличены там, где это требуется.
