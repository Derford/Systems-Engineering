# Контекст и границы системы

## System of Interest

**Система интереса:** маркетплейс вакансий и стажировок для университета.

Система отвечает за публикацию вакансий, отклики студентов, рекомендации, статусы, интервью и отчётность для карьерного центра.

## Внешние акторы

| Актор | Взаимодействие с системой |
|---|---|
| Студент | авторизуется, заполняет профиль, получает рекомендации, откликается |
| Работодатель | публикует вакансии, просматривает отклики, назначает интервью |
| Карьерный центр | контролирует вакансии, смотрит отчёты, анализирует трудоустройство |
| Модератор | проверяет вакансии, блокирует спам и некорректные публикации |

## Внешние системы

| Система | Назначение |
|---|---|
| SSO / ЛК студента | единый вход и получение базовых данных студента |
| E-mail / мессенджер | уведомления о статусах, интервью и откликах |
| Календарь | планирование интервью |
| Аналитическая система | выгрузка показателей трудоустройства |

## Границы ответственности

Входит в систему:

- каталог вакансий и стажировок;
- личный кабинет студента и работодателя;
- отклики и статусы;
- сервис рекомендаций;
- модерация вакансий;
- отчёты карьерного центра.

Не входит в систему:

- сама SSO-инфраструктура университета;
- внешние почтовые и мессенджер-сервисы;
- внешний календарь;
- юридическое оформление трудоустройства;
- фактическая работа студента у работодателя.
