# Компонентная схема сервисов

Ниже приведена общая UML Component Diagram для системы. Она дополняет C4 Component и показывает основные сервисы проекта на техническом уровне.

![Компонентная диаграмма](../diagrams/final/uml/14_component_diagram.svg)

<small>Общая компонентная схема сервисов маркетплейса.</small>


Компонентная схема показывает, что система состоит не из одного блока, а из набора связанных сервисов: API, сервис рекомендаций, уведомления, база данных и внешние интеграции.
