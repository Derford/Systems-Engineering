# Тест-кейсы

| TC-06 | Студент без профиля пытается откликнуться | система просит заполнить профиль |
| TC-07 | Система рассчитывает рекомендации | студент получает топ вакансий |
| TC-08 | Работодатель назначает интервью | создаётся событие и уведомление |
| TC-09 | E-mail недоступен | уведомление остаётся в очереди |
| TC-10 | Карьерный центр строит отчёт | отображаются агрегированные показатели |
