# Маркетплейс вакансий и стажировок для университета

**Авторы:** Ерёмин (м1-ИФСТ-12), Иванов (м1-ИФСТ-12), Петрушин (м1-ИФСТ-11)  
**Тип проекта:** учебный проект по системной инженерии  
**Предметная область:** EdTech / HRTech

## Идея проекта

Система представляет собой университетский маркетплейс вакансий и стажировок. Она объединяет студентов, работодателей и карьерный центр на одной платформе. Студенты получают рекомендации и откликаются на вакансии, работодатели публикуют предложения и назначают интервью, а карьерный центр видит аналитику по трудоустройству.

## Цель

Спроектировать информационную систему, которая поддерживает полный цикл работы с вакансиями: публикация, модерация, рекомендации, отклики, интервью, статусы и отчётность.

## Ключевые роли

| Роль | Интерес |
|---|---|
| Студент | найти стажировку или работу по профилю |
| Работодатель | получить релевантные отклики студентов |
| Карьерный центр | контролировать качество вакансий и статистику трудоустройства |
| Модератор | проверять вакансии и бороться со спамом |

## Основные функции

1. Публикация вакансии работодателем.
2. Отклик студента на вакансию.
3. Матчинговые рекомендации вакансий.
4. Назначение интервью и обновление статуса отклика.

## Стек, выбранный для проектирования

| Компонент | Технология |
|---|---|
| Frontend | React + TypeScript |
| Backend API | Spring Boot / REST |
| Recommendation Service | Python + FastAPI |
| База данных | PostgreSQL |
| Кэш | Redis |
| Уведомления | E-mail / мессенджер |
| Инфраструктура | Docker, GitHub Actions, GitHub Pages |

## Что важно в проекте

Проект сделан как учебная инженерная документация: есть требования, архитектура, UML, функциональные модели, BPMN, ADR, API и тест-кейсы. Диаграммы оформлены как PlantUML-исходники, чтобы их можно было воспроизвести и экспортировать в PNG/SVG.
