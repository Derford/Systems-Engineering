# Метрики рекомендаций

Рекомендательная система является важной частью проекта, поэтому качество рекомендаций выделяется как отдельный NFR-вызов.

## Основные метрики

| Метрика | Смысл | Целевое значение |
|---|---|---|
| Precision@5 | доля релевантных вакансий в первых 5 рекомендациях | ≥ 0.70 |
| Recall@10 | доля найденных релевантных вакансий в первых 10 | ≥ 0.60 |
| CTR рекомендаций | доля переходов по рекомендованным вакансиям | ≥ 12% |
| Apply Rate | доля рекомендаций, закончившихся откликом | ≥ 5% |
| Complaint Rate | доля жалоб на нерелевантные рекомендации | ≤ 3% |

## Какие признаки используются

- направление обучения;
- курс;
- навыки;
- опыт;
- тип занятости;
- город / удалённый формат;
- история откликов;
- требования вакансии.

## Объяснимость

Для студента система должна показывать простое объяснение: например, «вакансия подходит, потому что требуются Python и SQL, которые указаны в вашем профиле».
