# Техническое задание

## Назначение системы

Маркетплейс вакансий и стажировок предназначен для автоматизации взаимодействия между студентами, работодателями и карьерным центром университета.

## Основание для разработки

Учебный проект по системной инженерии. Тема выбрана из области EdTech/HRTech, так как она связана с реальными процессами университета и имеет понятные роли, данные, интеграции и нефункциональные требования.

## Цели разработки

- сократить время поиска стажировок студентами;
- повысить качество откликов для работодателей;
- дать карьерному центру прозрачную аналитику;
- снизить количество спама и нерелевантных вакансий.

## Основные функции

1. Авторизация через SSO.
2. Управление профилем студента.
3. Публикация и модерация вакансий.
4. Отклик на вакансию.
5. Рекомендации вакансий.
6. Управление статусами кандидатов.
7. Назначение интервью.
8. Формирование отчётов.

## Требования к данным

Система хранит данные о студентах, работодателях, вакансиях, откликах, интервью, рекомендациях, уведомлениях и событиях аудита.

## Ограничения

- система не заменяет внешний SSO;
- система не является кадровым документооборотом;
- рекомендации должны быть объяснимыми на базовом уровне;
- персональные данные должны обрабатываться с учётом приватности.
